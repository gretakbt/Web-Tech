hier das set-up:

https://docs.github.com/de/get-started/quickstart/set-up-git


Um repo zu klonen:
 "git clone https://github.com/gretakbt/Web-Tech.git"

Danach testen ob richtig konfiguriert (ab hier alles in dem Pfad ausführen) :
"git remote -v"
Wenn ja kommt das
"origin	https://github.com/gretakbt/Web-Tech.git (fetch)
origin	https://github.com/gretakbt/Web-Tech.git (push)"

Dann immer wenn Änderung:
erst commiten:
"git add .
git commit -m "Dein Commit-Beschreibung""

dann pushen (online zur verfügung stellen):
"git push origin main"


Bevor man anfängt zu arbeiten mögliche Änderungen pullen:

"git pull origin main"
